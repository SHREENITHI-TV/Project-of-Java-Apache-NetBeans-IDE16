# ❌⭕ Cross N Knot (Tic-Tac-Toe) - Run the JAR

This folder contains **CrossNKnot.jar** - a ready-to-run Java (Swing) Tic-Tac-Toe game.

---

## ✅ Quick Start (Run the JAR)

### 1) Install Java (Required)
You need **Java (JRE/JDK)** installed.

Check it:
```bash
java -version
```
If you see a version number, you’re good!!
